Frequently Asked Questions

How do I find a Discord User ID?

To find a Discord User ID:

  1. Enable Developer Mode in Discord Settings (User Settings → Advanced → Developer Mode)
  2. Right-click on any user's profile or username
  3. Select "Copy User ID" from the context menu

User IDs are 17-19 digit numbers that uniquely identify Discord accounts.

Why can't I download some avatars or banners?

There are several reasons why you might not be able to download certain media:

  • The user has deleted their account
  • The user has changed their avatar/banners recently and the CDN cache is updating
  • The user has a default avatar (which has limited resolution)
  • Discord's CDN is temporarily unavailable

If the issue persists, please contact our support team.
