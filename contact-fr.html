Foire aux questions

Comment trouver un ID utilisateur Discord ?

Pour trouver un ID utilisateur Discord :

  1. Activez le mode développeur dans les paramètres Discord (Paramètres utilisateur → Avancé → Mode développeur)
  2. Faites un clic droit sur le profil ou le nom d'utilisateur de n'importe quel utilisateur
  3. Sélectionnez « Copier l'ID utilisateur » dans le menu contextuel

Les ID utilisateur sont des nombres de 17 à 19 chiffres qui identifient de manière unique les comptes Discord.

Pourquoi ne puis-je pas télécharger certains avatars ou bannières ?

Il y a plusieurs raisons pour lesquelles vous pourriez ne pas pouvoir télécharger certains médias :

  • L'utilisateur a supprimé son compte
  • L'utilisateur a récemment changé d'avatar/bannières et le cache CDN est en cours de mise à jour
  • L'utilisateur a un avatar par défaut (qui a une résolution limitée)
  • Le CDN de Discord est temporairement indisponible

Si le problème persiste, veuillez contacter notre équipe de support.

Ce service est-il gratuit ?

Oui, notre service Visionneuse Discord PFP est complètement gratuit à utiliser. Nous ne facturons aucun frais pour accéder ou télécharger des photos de profil et des bannières Discord. Nous pouvons afficher des publicités pour soutenir le service, mais toutes les fonctionnalités de base restent gratuites.

Ce service est-il affilié à Discord Inc. ?

Non, notre service n'est pas affilié à, approuvé par ou parrainé par Discord Inc. Nous sommes un service indépendant qui utilise les API publiques et le CDN de Discord pour fournir des capacités de visualisation et de téléchargement d'images de profil.
